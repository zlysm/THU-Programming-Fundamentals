大数计算器实现了两种功能。

用户需要输入1或2来分别选择基本功能和进阶功能，选择完毕后，程序会提示用户输入相应表达式。

基本功能能够对加、减、乘、取余表达式进行计算，用户输入的数字必须是整数，且须在int范围内。数字之间不能有空格，数字与运算符之间可以有任意数量空格。若数字间有空格，那么程序会自动将两个数合并，即输入”12    34 + 5678“与”1234 + 5678“等效。

需要注意的是输入负数必须要加英文状态下的括号，但是处于开头的负数不需要加，即”1 + (-2)“，”-1 + 2“都是合法的。表达式应符合四则运算的规则，譬如用户不能对0取余，即像”1 % (-1 + 1)“是非法的。

进阶功能能够计算两个正整数的积，数字位数不应大于100000位。同样，数字之间不能有空格，数字与运算符之间可以有任意数量空格。若数字间有空格，那么程序会自动将两个数合并，即输入”12    34 * 5678“与”1234 * 5678“等效。

需注意进阶功能只能计算两个数的乘法，任何 '*' 以外的运算符将被判为非法。
